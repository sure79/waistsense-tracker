## 사용법
1. PC에서 터미널 열기 → `claude` 입력 (Claude Code 실행)
2. 아래 프롬프트 전체를 복사 → 붙여넣기 → Enter
3. Claude Code가 프로젝트를 자동 생성
4. `npm run dev` → localhost:3000에서 확인
5. GitHub 푸시 → Vercel 배포 → 아이폰 홈화면 추가

## 프롬프트 시작 ▼▼▼ (아래 전체를 Claude Code에 복사)

---

WaistSense 프로젝트 실행관리 PWA 웹앱을 Next.js 14 + TypeScript + Tailwind CSS로 만들어줘.

이 앱의 핵심 목적: "계획을 세우는 것"이 아니라 "실제로 행동하게 만드는 것".
사용자는 풀타임 직장인(평일 20~23시, 토요일 종일만 작업 가능).
"오늘 딱 1가지만 하면 된다"는 심리적 부담 최소화가 핵심 철학.

기술: Next.js 14 App Router + TypeScript + Tailwind + localStorage (DB 없음)
PWA 필수: manifest.json + service worker → 아이폰 홈화면 앱
알림 필수: Notification API → 매일 저녁 + 스트릭 끊김 방지 + 주간 리뷰

디자인: 메인 #1B998B(청록), 배경 #F8FFFE, 남색 #1B2A4A, 위험 #C44536, 성공 #27AE60
모바일 퍼스트, 하단 고정 탭 4개(🏠오늘/✅체크/📊현황/⚙️설정), safe-area-inset 적용

---

탭1 "오늘" (가장 중요한 화면, 앱의 90%):

시간대별 인사: 06~12시 "좋은 아침 ☀️" / 12~18시 "오늘도 고생 💪" / 18~21시 "퇴근 후 30분, 오늘의 미션 🔥" / 21~24시 "수고했어요, 내일을 위해 🌙"

D-Day 카운터 크게: "와디즈 오픈까지 D-XXX" (시작 2026-04-07, 목표 2026-09-22) + 진행률 바

🎯 오늘의 미션 카드 (핵심!):
현재 주차에서 미완료 첫 번째 항목 1개만 크게 표시.
"오늘 딱 이것만 하세요" 문구.
아래에 구체적 첫 행동 1줄 표시 (예: "mechasolution.com 접속 → ESP32-S3 검색 → 장바구니").
큰 "완료! ✅" 버튼 → 누르면 confetti(canvas-confetti) + navigator.vibrate(200) + 칭찬 메시지.
완료 시 다음 항목 자동 올라옴.
하루 2개 이상 완료 시 "오늘 2개나! 🎉" 보너스 칭찬.

연속 실행일 스트릭: "🔥 X일 연속!" 불꽃 크기 변화(1~7일 작은불, 8~14 중간, 15+ 큰불).
스트릭 끊기면 "괜찮아요, 다시 시작하면 됩니다" 격려.

투입 비용: "₩XX,XXX / 최대 ₩500,000" + 안전(초록)/주의(노랑)/경계(빨강) 프로그레스 바.

격려 메시지 (매일 랜덤 1개, 터치 시 다음):
"카카오 파스타도 처음엔 '이게 왜 필요해?'였습니다" /
"최악의 경우 잃는 돈 50만원. 최선의 경우 정부지원 8,000만원" /
"작동하는 데모가 있으면 협력자가 먼저 찾아옵니다" /
"6개월 뒤 프로토타입+앱+특허를 가진 창업자입니다" /
"내가 쓰려고 만든 것이 세계에서 가장 강력한 제품" /
"웰트도 삼성 직원 혼자 시작했습니다" /
"지방간 수치가 떨어지면 그게 최고의 사업계획서" /
"진짜 고객은 지방간 카페에 있습니다" /
"오늘 30분이면 됩니다. 딱 30분만"

이번 주 미니 프로그레스: "이번 주 3/5 완료 (60%)"

알림 시스템:
1. 매일 저녁: "🔥 오늘의 미션: [할일1개] — 딱 30분"
2. 주간 리뷰: "📊 이번 주 X/Y 완료! [격려]"
3. 스트릭 방지 (2일 미체크): "⚠️ 불꽃이 꺼지려고 해요! 1개만 하면 🔥X일!"
4. 마일스톤: "🎉 [마일스톤명] 달성!"

PWA: manifest standalone + service worker 캐싱 + apple-touch-icon(180x180 청록배경 WS) + splash screen.

심리 요소:
- 체크 시 confetti + vibrate + 에스컬레이션 칭찬 (1개→3개→5개 다른 메시지)
- 스트릭 불꽃 크기 변화 + 끊기면 "괜찮아요" 격려
- "지금까지 XX시간, ₩XX,XXX 투자. 여기서 멈추면 아까워요!"
- 밀린 주차: "순서보다 실행이 중요! 이번 주에 같이 해치우세요"
- 마일스톤 달성 시 미래 시각화 카드 ("프로토타입 차고 출근하는 나를 상상해보세요")

recharts 라이브러리로 차트. canvas-confetti로 축하. 모든 체크 상태 localStorage 저장.
